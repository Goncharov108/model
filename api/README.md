# API (заготовка, фаза 4)

Минимальный процесс на Node.js: `GET /health` возвращает JSON. Нужен для смок-теста в CI и как точка входа до отдельного NestJS-сервиса из канона-3.

```bash
cd api
npm start
# в другом терминале:
curl -s http://127.0.0.1:3847/health
```

Порт по умолчанию `3847`, переопределение: `PORT=4000 npm start`.

Черновик контракта: `openapi.yaml` в этом каталоге.
