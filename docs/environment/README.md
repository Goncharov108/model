# Документация окружения (репозиторий model)

Единый индекс по инфраструктуре, деплою и стеку. **Без секретов** — пароли, ключи и токены только в `private/`, GitHub Secrets или на сервере (закон 1 SWOD).

| Файл | Содержание |
|------|------------|
| [STACK.md](./STACK.md) | Слои: прод / репозиторий / запланировано (канон-3) |
| [GITHUB.md](./GITHUB.md) | Репозиторий, ветка, CI |
| [DNS_REG_RU.md](./DNS_REG_RU.md) | Домен, DNS, SSL |
| [SERVER.md](./SERVER.md) | Хост, ОС, SSH, пути |
| [DEPLOY.md](./DEPLOY.md) | Чеклист выката v1 |

**UI:** Мастер-админ → вкладка «Окружение» (`/master-admin/environment`), persist `model-environment-docs-v1`, экспорт/импорт JSON version 1.

**Зафиксировано владельцем (2026-05):** GitHub `Goncharov108/model` (private), домен `live-model.ru`, сервер пока не подключён.
