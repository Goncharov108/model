## API за nginx

Прокси `location /api/` → `http://127.0.0.1:3847` (PORT из `api/server.mjs` или `process.env.PORT`).

## Firewall (рекомендация)

- Открыть: 80, 443
- SSH: по возможности только с доверенных IP (уточнить у владельца)
