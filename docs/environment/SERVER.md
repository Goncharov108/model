# Сервер

**Без паролей и приватных ключей в этом файле.**

## Текущее состояние (2026-05)

- Прод-сервер для `live-model.ru` пока не зафиксирован в этом репозитории.
- Рабочий Linux-сервер Hermes (для автоматизаций) существует отдельно и не является продом проекта `model`.
- Поэтому IP/доступы/юниты прода здесь не публикуются до финального подключения.

## Целевые параметры прода (v1)

- ОС: Ubuntu 22.04 LTS
- SSH-пользователь: `deploy`
- Node: 22 (см. `.nvmrc`)

## Пути деплоя (v1)

- Статика SPA (`web/dist`): `/var/www/model/web`
- API (`api/server.mjs`): `/opt/model/api`

## API за nginx

Прокси `location /api/` → `http://127.0.0.1:3847`

## Firewall (рекомендация)

- Открыть: `80`, `443`
- SSH: по возможности только с доверенных IP

## Что обновить после подключения прода

- IP/FQDN сервера
- Имя systemd unit для API (например `model-api.service`)
- Фактический путь к nginx-конфигу
