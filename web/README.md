# Веб-интерфейс (локально)

Канонические решения и контекст проекта: корень репозитория, `SWOD_ZAKONOV.txt` (закон 6).

## Команды

```bash
cd web
npm install
npm run dev
```

Открой в браузере адрес из вывода Vite (обычно `http://localhost:5173/`).

```bash
npm run build
npm run preview
```

## Node

В корне репозитория есть `.nvmrc` (сейчас `22`). Убедись, что в `PATH` используется соответствующий Node (например через nvm).
