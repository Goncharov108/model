# Документация и идеи model

**Документация по репозиторию и продукту** — идеи фич, UX, дополнения к планам, заметки к `PLAN_REALIZATSII.md`, связи с UI.

- Факты и операционка без секретов: `docs/` в корне репо, `docs/environment/`
- Мастер-админ в браузере: вкладки Обзор, Окружение, Пользователи, Продвинутые

Имя репо кода: **model**. «Мастер» — продуктовая рамка из канонов.
