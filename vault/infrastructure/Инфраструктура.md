# Инфраструктура (заметки)

Черновики к **документации model** по серверу, DNS, деплою. Рядом с личными пометками в `journal/`, но с упором на связь с репо.

**Канон фактов (без секретов):** `docs/environment/`.  
**Секреты:** только `private/`.

Эта папка — мост между головой и `docs/environment/` + вкладкой **Окружение**.
